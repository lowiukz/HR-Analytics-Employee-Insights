# HR Analytics: Employee Turnover Analysis with SQL & Power BI

## 📌 Project Overview

This project analyzes employee turnover using a multi-table HR dataset containing approximately **300,000 employee records**.

The goal is to identify patterns behind employee turnover by performing SQL-based analysis and presenting the results through an interactive Power BI dashboard.

---

## 🎯 Business Problem

**Why are employees leaving the company?**

To answer this question, I analyzed different dimensions of employee turnover, including department, job title, tenure, salary, and gender.

---

## 🗂 Dataset

- Source: [Kaggle](https://www.kaggle.com/datasets/isissantoscosta/365ds-practice-exams-people-analytics-dataset/data)
- Approx. 300,024 unique employee records
- Multi-table relational dataset

Main tables:

- Employees
- Departments
- Department Employees
- Department Manager
- Titles
- Salaries

---

## 🔍 Key Insights

- Overall employee turnover rate is **19.97%**.
- Turnover rates are relatively similar across departments.
- Manager positions showed no employee exits.
- Employees with **7+ years of tenure** had the highest turnover rate.
- Average salaries between male and female employees were very similar.
- Sales had the highest average salary, while HR had the lowest, yet both departments had similar turnover rates, suggesting salary alone may not explain employee turnover.
